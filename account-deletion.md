앱: **이룸(iroom)** · 개발자: **iroom**

이룸은 사용자 데이터를 자체 서버에 저장하지 않습니다(사용자 소유 클라우드, BYOC). 계정 및 데이터는 사용자가 앱에서 직접 삭제할 수 있습니다.

## 계정 연결 해제 및 데이터 삭제 방법

1. **클라우드 데이터 삭제**: 앱 실행 → **설정 → 클라우드 동기화 → "클라우드에서 데이터 삭제"** 를 누르면, 사용자 본인 Google Drive의 앱 전용 폴더에 저장된 데이터가 삭제됩니다.
2. **로그아웃(계정 연결 해제)**: 앱 → **설정 → 클라우드 동기화 → 로그아웃** 시 기기에 저장된 인증 토큰이 즉시 폐기되고, Google Drive 접근 권한(OAuth) 철회를 시도합니다.
3. **기기 데이터 삭제**: 앱을 삭제하면 기기에 저장된 모든 데이터(습관·완료기록·집중 세션·일기·챌린지·설정)가 함께 삭제됩니다.

## 삭제되는 데이터

- 습관, 완료 기록, 집중 세션, 일기, 챌린지, 앱 설정 등 모든 서비스 데이터
- 기기에 저장된 인증 토큰

## 보관되는 데이터

- 없음. 이룸은 사용자 데이터를 자체 서버에 저장·보관하지 않으므로, 위 절차 이후 개발자 측에 남는 데이터가 없습니다.
